1. Install all dependencies

4. Start backend (port 5000)
5. Start frontend (port 3000)
6. Open in browser

## 📖 Manual Setup

### Step 1: Install Dependencies

**Terminal 1 - Backend:**
```cmd
cd backend
npm install
```

**Terminal 2 - Frontend:**
```cmd
cd frontend
npm install
```

### Step 2: Start MongoDB
Make sure MongoDB is running on your system

### Step 3: Seed Database
```cmd
cd backend
npm run seed
```

### Step 4: Start Servers

**Terminal 1 (Backend):**
```cmd
cd backend
npm run dev
```

**Terminal 2 (Frontend):**
```cmd
cd frontend
npm run dev
```

### Step 5: Open Browser
Go to: http://localhost:3000
