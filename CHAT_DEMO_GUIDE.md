# 🤖 AI Chat Feature - Demo Guide

## How to Demo the Chat Feature

### Step 1: Access Chat
1. Login to your account
2. Click "AI Chat" in the sidebar
3. You'll see a welcome message

### Step 2: Try These Questions

#### Basic Queries
```
"Hello"
"Hi, what can you do?"
```
**Response**: Welcome message with capabilities

#### Eligibility Check
```
"What scholarships am I eligible for?"
"Show me scholarships I can apply to"
"Can I apply for any scholarships?"
```
**Response**: List of eligible scholarships based on your profile

#### Deadline Queries
```
"Show me closing soon scholarships"
"Which scholarships have urgent deadlines?"
"What's closing this week?"
```
**Response**: Scholarships closing within 7 days

#### High Amount
```
"Show me high amount scholarships"
"What are the highest paying scholarships?"
"Show me scholarships above 50000"
```
**Response**: Scholarships with amount ≥ ₹50,000

#### Document Requirements
```
"What documents do I need?"
"What documents are required?"
"Tell me about required documents"
```
**Response**: Common document list

#### Application Help
```
"How to apply for scholarships?"
"What's the application process?"
"How do I apply?"
```
**Response**: Step-by-step application guide

#### Income-Based Queries
```
"I have 2L income, what can I apply?"
"My income is 500000, show scholarships"
"Income 3 lakh scholarships"
```
**Response**: Scholarships matching that income level

#### Education-Based
```
"I'm a diploma CSE student"
"Show me scholarships for engineering"
"I'm doing computer science"
```
**Response**: Technical/engineering scholarships
